Vocab: AI Terms To Know
Glossary · Claude Edition
Prompt
The instruction you give Claude. Specificity and context = better output. Think of it as a creative brief.
Chat
A single conversation. Claude remembers everything within it, but starts fresh in a new chat unless you're in a Project.
Project
A persistent workspace that retains context across chats. Use for ongoing work so Claude always knows your background.
Artifact
A standalone output — doc, code, prompt — that lives outside the chat thread and can be directly edited.
Skills
Instruction sets that teach Claude to handle recurring tasks consistently. Powers our BE Brand QA Agent.
Memory
Claude's ability to retain facts across conversations. Persistent in Projects; resets in standalone chats.
MCP
Model Context Protocol — lets Claude connect to external tools like Asana, Drive, or Slack to take real actions.
Hallucination
Confident but incorrect AI output. Always verify before using in briefs, copy, or client-facing materials.
AI Agent
AI that executes multi-step workflows autonomously. Our IMC and Brand QA tools are both agents.
Claude Products: What's What

Everyday Use
Claude.ai Chat
Web and mobile interface for writing, research, brainstorming, and drafting. Works in Projects for persistent context. Most of the team uses this daily.
💻
For Developers
Claude Code
Command-line tool that lets Claude write, edit, and run code in your codebase. Best for engineering teams building automations and apps.
🖥️
Rolling Out Late March 2026
Claude Cowork
Desktop tool for non-developers. Automates file and task management — move files, fill spreadsheets, organize folders. No code required.
💡
Pro Tip: Branching Chats in Claude
Like ChatGPT's branch feature — Claude has it too. Hover over a previous response → click the edit (pencil) icon on your message → rewrite your prompt → Claude generates a new branch from that point. Your original thread stays intact. Great for testing different tones or angles without starting over.
